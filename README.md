# 目录说明
本项目主要基于前端技术栈实现。  
* components：可视化组件
* doc：关于麻将的文档
* js:主要逻辑
* majiang：使用python探索麻将
* pages:网页
* public:公共资源
* res:声音资源
* test:测试代码
